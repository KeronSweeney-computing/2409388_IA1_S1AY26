Storyboard & UI Planning

Mobile Layout (min 600px)

Header: Logo + site title (brand).
Navigation: Links wrap to the next line if needed.
Hero: Heading + short introduction + call-to-action buttons.
Profile: Circular profile image + demographic details.
Content sections: Vision & Mission, Personality cards, Mentor/Hero section, AI Report summary, Twelve Pillars link.
Favourites: Table section (scrolls horizontally if needed).
Contact: Email + GitHub + internal bookmark link (#contact).
Footer: Copyright and short site info.

Desktop Layout (min 950px)

Header: Logo/title on the left, navigation links on the right (single row).
Hero split grid: Left side contains heading and introduction; right side contains the profile card and image.
Cards: Personality section displays as a 3-column grid.
Panels: Vision/Mission and Mentor/Hero sections display as panel blocks with text + image (split layout).
Favourites: Table is centered and stays within a max width for a clean look.
Footer: Full width footer with muted text.

Div Groups Used (UI Structure)

Main container: div.container centers content.
Navigation bar: header.site-header and nav.navlinks.
Hero layout: section.hero and div.hero-grid.
Sections: section.section with header.section-head.
Cards: div.grid-3 containing article.card.
Split layout: div.split (text + image).
Table wrapper: div.table-wrap to control width and responsiveness.

Icons and Dimensions

  • Logo style: Square badge with rounded corners
  • Logo size: 44px × 44px
  • Profile image: Circular crop
  • Profile image size: 176px × 176px
  • Section images: Landscape format inside .media container
  • Recommended section image size: 1200px × 800px (or similar ratio)

CSS Decisions

Font-family: system-ui
Root font-weight: 400–850
Border radius: 18px
Spacing: 10–44px

Colour Palette (with RGB)

Background: rgb(7, 22, 43)
Panels/Cards: rgb(11, 35, 68)
Accent (Links/Buttons): rgb(65, 201, 255)
Accent Light: rgb(139, 225, 255)
Text: rgb(255, 255, 255)
